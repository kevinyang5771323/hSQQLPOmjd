# 人脸表情识别系统 python825

## 项目概述


## 技术栈

- **GUI框架**：PyQt5
- **图像处理**：OpenCV
- **深度学习**：TensorFlow
- **神经网络**：DenseNet121
- **人脸检测**：face-recognition
- **数据处理**：NumPy
- **图像显示**：Pillow

## 功能模块

### 1. 主窗口模块
- 系统启动后的主交互界面。

### 2. 图片识别模块
- 对用户提供的静态图片进行表情识别。

### 3. 视频分析模块
- 对视频文件进行逐帧表情分析。

### 4. 实时摄像头模块
- 实时捕捉摄像头输入，并进行表情识别。

### 5. 表情分类模块
- 将识别出的表情归类，支持七种常见表情分类。

### 6. 情绪概率可视化模块
- 以可视化方式展示各表情分类的概率。

### 7. 表情图标显示模块
- 在界面上以图标形式直观显示当前表情。

### 8. 置信度分析模块
- 显示当前表情识别的置信度。
